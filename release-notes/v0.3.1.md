# Another OpenCode Plugin for Obsidian v0.3.1

This patch release rebuilds the plugin with the fixes that landed after
`v0.3.0`.

## Fixes

- Fixed prompt-context path normalization on Windows.
  - Parent-directory checks now use the current platform path separator.
  - This keeps vault-relative prompt context paths consistent across macOS,
    Linux, and Windows.
- Kept the release checks aligned with the packaged Obsidian assets.

## Installation

BRAT users can install or update from:

```text
jensenojs/another-opencode-for-obsidian
```

The release includes:

- `manifest.json`
- `main.js`
- `styles.css`
