# Another OpenCode Plugin for Obsidian v0.3.3

This patch release fixes two context-surface issues found after `v0.3.2`.

## Fixes

- Removed stale one-shot selected-text cards after a prompt is sent.
  - Merged native context cards now keep a stable projection identity when the
    workspace cursor line moves near the selected range.
  - This lets the successful-send cleanup remove the selected-text card from
    the OpenCode composer.
- Made the Obsidian context status popover dismiss reliably.
  - The popover now owns a transparent outside-click layer in the Obsidian
    window.
  - Clicking outside the popover, including over the OpenCode iframe area,
    closes it.

## Installation

BRAT users can install or update from:

```text
jensenojs/another-opencode-for-obsidian
```

The release includes:

- `manifest.json`
- `main.js`
- `styles.css`
