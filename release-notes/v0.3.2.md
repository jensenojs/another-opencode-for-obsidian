# Another OpenCode Plugin for Obsidian v0.3.2

This patch release fixes several integration issues found after `v0.3.1`.

## Fixes

- Restored native prompt context cards after an upstream OpenCode Web UI bundle
  change.
  - The prompt context port patch now tolerates the current minified provider
    accessor shape.
  - This restores automatic attach for Obsidian-owned prompt context cards.
- Reduced visual noise from automatic context cards.
  - Nearby same-file native file cards now merge when their line ranges overlap
    or their gap is at most 50 lines.
  - Workspace active-line cards can merge into nearby selected-text cards.
- Preserved OpenCode Web UI file-row interactions.
  - File and tool headers now keep OpenCode's own click behavior, including
    accordion toggles and tool subtitle actions.
  - Obsidian vault navigation is still available from concrete diff lines and
    markdown vault path text.

## Installation

BRAT users can install or update from:

```text
jensenojs/another-opencode-for-obsidian
```

The release includes:

- `manifest.json`
- `main.js`
- `styles.css`
